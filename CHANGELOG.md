# 版本更新日志 (Changelog)

## v0.2.1 (当前版本 - Current)
**提交 ID:** `9fca028`
**日期:** 2026-02-11

### 🚀 主要变更
1.  **修复 (Fix):** 修复了首页 "查看完整预报报告" (Avalanche Forecast) 按钮点击无效的问题，现在可以正确跳转到详情页。
2.  **数据同步 (Refactor):** 解决了首页和观测列表页 (Observation List) 数据不一致的问题。重构了 API 层，实现了在无后端连接时的智能模拟数据 (Mock Data) 回退机制，确保两端看到的数据是一致的。
3.  **UI 优化 (Polish):** 优化了雪崩玫瑰图 (Rose Diagram) 的图例样式，使其更符合新的设计规范。
