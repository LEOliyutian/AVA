# 可用斜杠命令速查

直接在 Claude Code 对话框输入命令即可，不需要打开任何文件。

---

| 命令 | 别名 | 干什么 | 示例 |
|------|------|--------|------|
| `/be` | `/backend-engineer` | 后端工程师：设计接口、数据库、服务层 | `/be 设计用户导出接口` |
| `/pm` | `/product-manager` | 产品经理：分析需求、写 PRD、评审流程 | `/pm 预报审核工作流怎么设计` |
| `/ui-designer` | — | UI 设计师：设计页面、评审界面 | `/ui-designer 帮我看看仪表板布局` |
| `/debug` | `/full-stack-debugger` | 调试专家：排查 bug、定位问题 | `/debug 页面空白没有数据` |

---

## 什么时候用哪个

- **要做新功能** → 先 `/pm` 想清楚需求，再 `/be` 设计接口
- **接口 / 数据库问题** → `/be`
- **页面长得不好看 / 体验有问题** → `/Leo-UI`
- **出 bug 了** → `/debug`
- **不知道用哪个** → 直接说问题，Claude 自己判断
