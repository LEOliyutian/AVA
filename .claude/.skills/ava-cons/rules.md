# Avalanche Science Rules

All avalanche related content must follow established snow science.

---

# 1 Terminology Accuracy

Use internationally recognized terminology.

Example:

Correct:

- Slab avalanche
- Weak layer
- Persistent weak layer
- Temperature gradient
- Faceting

Avoid informal language.

---

# 2 Snowpack Mechanics

Snowpack explanations must respect physical processes:

Important mechanisms:

• Temperature gradient metamorphism  
• Rounding  
• Faceting  
• Melt-freeze cycles  

---

# 3 Avalanche Release Requirements

Avalanche formation requires:

1. Slab
2. Weak layer
3. Failure propagation
4. Slope angle

If one element is missing, slab avalanche cannot occur.

---

# 4 Avalanche Risk Communication

Danger levels must follow **EAWS scale**:

1 Low  
2 Moderate  
3 Considerable  
4 High  
5 Extreme  

Risk messaging must clearly separate:

• Likelihood  
• Size  
• Distribution  

---

# 5 Snowpack Tests Interpretation

Tests include:

• Compression Test (CT)
• Extended Column Test (ECT)
• Propagation Saw Test (PST)

Interpret results carefully.

Example:

CT11 Q1 = sudden planar failure.

This suggests:

High fracture propagation potential.

---

# 6 Terrain Factors

Avalanche terrain must consider:

• slope angle
• aspect
• elevation
• terrain traps
