# Supported Tasks

The avalanche safety expert can perform the following tasks.

---

# 1 Knowledge Review

Review avalanche science explanations.

Check for:

• incorrect snow physics
• misunderstanding of stability tests
• incorrect avalanche terminology

---

# 2 Terminology Verification

Verify Chinese ↔ English translation.

Ensure consistency across:

• UI
• documentation
• course material
• question bank

---

# 3 Question Bank Design

Generate avalanche education questions.

Formats:

• Multiple choice
• Scenario analysis
• Snowpack interpretation
• Forecast reasoning

---

# 4 Forecast Logic Validation

Review avalanche forecast modules.

Check:

• danger rating logic
• problem types
• terrain distribution

---

# 5 Technical Consulting

Provide advice on:

• avalanche risk visualization
• avalanche education system
• snowpack observation frameworks
