# Avalanche Terminology Standard

This document defines the **official terminology mapping** used in the project.

---

# Snow Science Terms

Snowpack = 雪层结构

Weak Layer = 弱层

Persistent Weak Layer = 持久弱层

Temperature Gradient = 温度梯度

Faceting = 结晶化

Rounding = 圆化

---

# Avalanche Terms

Slab Avalanche = 板层雪崩

Loose Snow Avalanche = 松散雪崩

Storm Slab = 风暴板层

Wind Slab = 风板

Persistent Slab = 持久板层

Deep Persistent Slab = 深层持久板层

---

# Stability Tests

Compression Test (CT) = 压缩测试

Extended Column Test (ECT) = 延伸柱测试

Propagation Saw Test (PST) = 锯切传播测试

---

# Failure Types

Sudden planar (Q1) = 突然平滑断裂

Sudden collapse (Q1) = 突然塌陷断裂

Progressive compression (Q2) = 渐进压缩断裂

Resistant planar (Q3) = 阻力型断裂
